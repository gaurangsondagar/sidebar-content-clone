Sidebar Content Clone is a WordPress plugin that allows you to clone all widgets from one sidebar area to another sidebar area by one click.


== Description ==

Sidebar Content Clone is good handy and free solution for clone all widgets from one sidebar area to another area sidebar by one click.

You have to select source sidebar and destination sidebar that you want to clone and then click "Clone Widgets" button.

Sidebar Content Clone Plugin Features

1. Clone All Widgets from one sidebar to another sidebar
2. Translation Ready
3. Cross Browsers Support ( Firefox, Chrome, Safari, Opera, etc. )

= Technical Support =
We are ready for any support issues and feature suggestions. Please contact us on [support forum](http://wordpress.org/support/plugin/sidebar-content-clone)

== Installation ==

1. Upload the **sidebar-content-clone.zip** file Via WordPress Admin > Plugins > Add New,
2. Alternately, upload **sidebar-content-clone** folder to the /wp-content/plugins/ directory via FTP,
3. Activate the **Sidebar Content Clone** plugin from Admin > Plugins.

== Frequently asked questions ==
= Does Sidebar Content Clone allow cloning all widgets from one sidebar to another? =
Yes, Sidebar Content Clone allows you to clone all widgets from one sidebar to another, including all their settings.

= Where can I get support for Sidebar Content Clone? =
You can ask for help with the Sidebar Content Clone plugin on the [Support Forum](http://wordpress.org/support/plugin/sidebar-content-clone).

= Is Sidebar Content Clone compatible with the Gutenberg widget editor? =
Yes, Sidebar Content Clone is compatible with the Gutenberg-based widget editor and supports cloning sidebar widget areas within it.
